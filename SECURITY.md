## Reporting a Vulnerability

Please report any found security vunerabilites in V2X Hub releases using the [Issues](https://github.com/usdot-fhwa-OPS/V2X-Hub/issues) tab. These should be reported as **Released Bugs** and please add the issue label **Vulnerability**.
